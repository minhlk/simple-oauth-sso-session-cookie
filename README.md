## Introduction
Demo the different between ways to authentication, authorization by using session-cookie, jwt, SSO 

## How to
### 0. Install dependencies

```sh
npm install
```

### 1. Create .env file base on .env.example

```sh
cp .env.example .env
```

### 2. Session-cookie

```sh
npm run start-session
```

### 3. JWT

```sh
npm run start-jwt
```

### 4. SSO
```sh
npm run start-sso
```

### Access localhost
http://localhost:3030/
